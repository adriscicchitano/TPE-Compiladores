El archivo ejecutable .jar se encuentra en la carpeta code/test_cases.
Para ejecutarlo, se debe abrir una consola, navegar hasta test_cases y ejecutar

java -jar g21_compiler.jar [PATH al codigo a compilar]

Luego de la compilacion, se obtendran como salida varios archivos, los cuales incluyen
la tabla de símbolos, errores, warnings, tokens y estructuras sintácticas de codigo.
Estos archivos se crean dentro de la carpeta donde se ejecuta el comando anterior.
